# RigAI 🤖✨
Trợ lý AI thông minh, thân thiện – trả lời mọi thứ bằng tiếng Việt với độ chính xác cao.

RigAI là một hệ thống trí tuệ nhân tạo được xây dựng trên nền tảng ngôn ngữ lớn (LLM) hiện đại, tập trung vào khả năng giao tiếp tự nhiên và hỗ trợ con người trong công việc, học tập và cuộc sống hàng ngày. Với sứ mệnh trở thành người bạn đồng hành thực sự, RigAI ưu tiên sự chân thành, chính xác và dễ hiểu trong mọi câu trả lời.

# 🎯 Các tính năng nổi bật

🗣️ Giao tiếp tiếng Việt mượt mà: hiểu và trả lời tiếng Việt tự nhiên, phù hợp ngữ cảnh.

📝 Định dạng Markdown thông minh: trình bày câu trả lời rõ ràng với in đậm, gạch đầu dòng, bảng biểu, code block…

🔍 Tra cứu kiến thức chuyên sâu: hỗ trợ giải thích các khái niệm khoa học, toán học, lập trình, văn hóa, lịch sử…

🧠 Xử lý logic mạnh mẽ: có khả năng suy luận, phân tích và đưa ra giải pháp chi tiết.

⚡ Phản hồi nhanh, thân thiện: giọng điệu tích cực, sẵn sàng giúp đỡ.
